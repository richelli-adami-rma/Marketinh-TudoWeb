📊 Pesquisa de Atendimento ao Cliente em Python
📌 Descrição

Este projeto consiste em um programa simples em Python que realiza uma pesquisa de satisfação de atendimento ao cliente.

O sistema coleta informações de 50 entrevistados, incluindo:

Nome
Idade
Opinião sobre o atendimento

As opções de avaliação são:

1 - Excelente
2 - Bom
3 - Ruim

Ao final, o programa exibe a quantidade total de respostas:

✅ Excelente
❌ Ruim
🎯 Objetivo

Demonstrar o uso de:

Estrutura de repetição for
Estruturas condicionais if/elif
Entrada e saída de dados (input e print)
Contadores em programação
💻 Tecnologias Utilizadas
Python 3
